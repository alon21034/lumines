### Lumines ###

##### Install

```
brew install mongodb
npm install express mongodb body-parser jade
```

##### Usage

```
node server.js
```
Connect to [http://localhost:3000](http://locathost:3000/)
```
mongo
$ db.lumines_score.find()
```

##### Deploy

```
git push heroku product:master
```
